## 特性

- **边下边播** — 搜索种子后立即开始观看，无需等待下载完成
- **高画质播放** — GPU 加速视频渲染，搭配 Anime4K 实时超分辨率，画面更清晰
- **多种片源** — 通过 Bangumi 浏览动漫信息，从在线站点或种子站搜索资源
- **追番管理** — 追番列表、观看历史自动续播、新番日历追踪最新剧集

## 安装

从 [GitHub Releases](https://github.com/Twacqwq/Kuriume/releases/latest) 下载最新版本。

| 平台 | 系统要求 |
|------|----------|
| macOS    | macOS 11+ |
| Windows  | Windows 10+ |
| iOS      | iOS 16+ |

各平台应用商店的分发正在计划中。

欢迎贡献！请参阅 [CONTRIBUTING.md](CONTRIBUTING.md) 了解开发环境搭建、构建方式与贡献指南。
